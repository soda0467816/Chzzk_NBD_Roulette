# 난바다의 룰렛

치지직 라이브 방송의 채팅과 후원을 실시간으로 모아서 바로 룰렛으로 돌리는 Windows 데스크톱 앱입니다.

치지직 버튜버 [난바다](https://chzzk.naver.com/b1ac916d727d32cb7f472b50f6f97654) 님의 아이디어를 기반으로 만들어졌습니다.

## 설치 방법

1. [Releases](https://github.com/soda0467816/Chzzk_NBD_Roulette/releases) 페이지로 이동합니다.
2. 최신 버전의 `Nanbada_Roulette_0.1.0_x64-setup.exe` 파일을 다운로드합니다.
3. 설치 파일을 실행한 뒤 안내에 따라 설치합니다.
4. 설치가 끝나면 `난바다의 룰렛`을 실행합니다.

태그 페이지는 버전 확인용이고, 실제 설치 파일 다운로드는 **Releases 페이지**에서 하는 것을 권장합니다.

## 핵심 기능

- 방송 주소 확인 후 실시간 수집 시작
- 실시간 수집 대상 선택
  - 도네만
  - 채팅만
  - 도네 + 채팅
- 채팅 필터링 / 도네 필터링
  - 각각 켜고 끌 수 있음
  - 여러 개 입력 가능
- 도네 유형 필터
  - 일반 도네
  - 영상도네
  - 미션도네
- 묶기 기준
  - 닉네임
  - 메시지
  - 닉네임 + 메시지
- 집계 결과 탭
  - 전체
  - 도네
  - 채팅
  - 필터 결과
- 필터 결과를 룰렛 대상으로 이관
- 저장된 룰렛 관리
- 최근 당첨 기록
- 연속 회전 / 스킵 / 전체화면 룰렛
- 자동 종료 시간 설정
  - 카운팅 시간
  - 몇 시까지

## 현재 동작 방식

이 앱은 OAuth 로그인 없이 동작합니다.

- 사용자가 치지직 라이브 주소를 입력합니다.
- 방송이 실제로 켜져 있는지 확인합니다.
- 비공식 실시간 수집 경로로 채팅/후원 이벤트를 받습니다.
- 받은 이벤트를 현재 설정에 따라 집계하고 룰렛에 반영합니다.

## 집계 규칙

### 도네

`1개 단위 금액` 기준으로 칸 수를 계산합니다.

예:

- 1개 단위 금액: 1000원
- 후원 금액: 3000원
- 결과: 3칸

### 채팅

- 금액 표시는 0원
- 채팅 1회당 1칸으로 계산

## 필터링

### 채팅 필터링

- 켜면 입력한 값으로 시작하는 채팅만 반영합니다.
- 여러 개 입력 가능
- 끄면 채팅 전체를 수집합니다.

예:

- `!`
- `룰렛`
- `바하`

### 도네 필터링

- 켜면 입력한 값으로 시작하는 도네 메시지만 반영합니다.
- 여러 개 입력 가능
- 끄면 모든 도네를 반영합니다.

### 도네 유형 필터

- 일반 도네
- 영상도네
- 미션도네

미션도네는 성공으로 판단된 경우만 집계 반영을 시도합니다.

## 필터 결과 탭

집계 결과의 `필터 결과` 탭에서 별도 후보 집합을 만들 수 있습니다.

지원 조건:

- 대상: 전체 / 도네 / 채팅
- 금액: 사용 안 함 / 이상 / 이하 / 정확히 / 구간
- 키워드: 사용 안 함 / 포함 / 시작 / 정확히

가능한 동작:

- 조건에 맞는 항목만 따로 보기
- `룰렛으로 이관`
  - 현재 룰렛 대상을 필터 결과 기준으로 전환
- `전체 집계 복귀`
  - 다시 전체 집계 기준으로 복귀

## 시간 제한

시간 제한은 둘 중 하나만 선택할 수 있습니다.

- 카운팅 시간
- 몇 시까지

둘 다 끄는 것은 가능합니다.

수집 중이면 메인 화면의 현재 집계 영역에서

- 남은 시간
- 종료 기준

을 바로 확인할 수 있습니다.

종료되면 기준 문구가 `종료됨`으로 바뀝니다.

## 저장

주요 설정과 저장본은 앱 데이터 파일에 저장됩니다.

저장되는 내용:

- 설정
- 저장된 룰렛
- 최근 당첨 내역
- 필터 결과 상태

실시간 raw 로그와 현재 세션의 실시간 이벤트는 메모리에만 유지됩니다.

## 배포

사용자에게는 설치 파일 하나만 전달하면 됩니다.

- 설치 파일: `난바다의 룰렛_0.1.0_x64-setup.exe`

사용자는 Node.js, Rust, Cargo, Tauri CLI를 따로 설치할 필요가 없습니다.

## 프로젝트 구조

- `src/pages/streamer.html`
  - 메인 화면 구조
- `streamer-app.js`
  - 메인 로직
- `styles.css`
  - 메인 스타일
- `lib/chzzk-live-donation-client.mjs`
  - 치지직 실시간 채팅/후원 수집기
- `server.mjs`
  - 개발용 브리지 서버
- `src-tauri/src/main.rs`
  - Tauri 진입점
- `src-tauri/src/commands/unofficial_live.rs`
  - 실시간 collector 제어
- `src-tauri/src/commands/app_state.rs`
  - 앱 상태 저장/불러오기
- `src-tauri/src/storage/app_state_store.rs`
  - 앱 데이터 JSON 저장소
- `src-tauri/capabilities/default.json`
  - Tauri 이벤트 권한 설정

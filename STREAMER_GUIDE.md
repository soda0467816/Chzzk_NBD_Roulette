# 난바다의 룰렛 사용 가이드

이 문서는 스트리머가 방송 중 실제로 어떤 순서로 누르면 되는지와, 각 기능이 무엇을 하는지 빠르게 확인하기 위한 안내서입니다.

치지직 버튜버 [난바다](https://chzzk.naver.com/b1ac916d727d32cb7f472b50f6f97654) 님의 아이디어를 기반으로 만들어졌습니다.

## 기본 사용 순서

1. 앱 실행
2. 방송 주소 입력
3. `방송 확인`
4. 운영 패널에서 수집 대상과 필터링 설정 확인
5. `실시간 시작`
6. 최근 후원 / 집계 결과 확인
7. `돌리기`

## 방송 주소

메인 화면의 방송 주소 칸에 치지직 라이브 주소를 입력합니다.

예:

```text
https://chzzk.naver.com/live/xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

`방송 확인`을 누르면:

- 주소 형식 확인
- 현재 라이브 방송인지 확인
- 연결 대상 채널 확인

을 먼저 진행합니다.

## 운영 패널에서 바꾸는 옵션

운영 패널에서는 이런 설정을 바꿀 수 있습니다.

- 실시간 수집 대상
  - 도네만
  - 채팅만
  - 도네 + 채팅
- 채팅 필터링 사용
- 채팅 필터링 값
- 도네 필터링 사용
- 도네 필터링 값
- 도네 유형 필터
- 묶기 기준
- 1개 단위 금액
- 룰렛 회전 시간
- 자동 종료 시간

## 채팅 필터링

`채팅 필터링 사용`을 켜면 입력한 값으로 시작하는 채팅만 반영합니다.

예:

- `!`
- `룰렛`
- `바하`

여러 개를 추가할 수 있고, 하나라도 앞에서 맞으면 통과합니다.

끄면 채팅 전체를 수집합니다.

## 도네 필터링

`도네 필터링 사용`을 켜면 입력한 값으로 시작하는 도네 메시지만 반영합니다.

여러 개를 추가할 수 있습니다.

끄면 모든 도네를 반영합니다.

## 도네 유형 필터

선택 가능한 항목:

- 일반 도네
- 영상도네
- 미션도네

미션도네는 성공으로 판단된 경우만 집계 반영을 시도합니다.

## 묶기 기준

- 메시지만 켜짐
  - 메시지 기준으로 묶음
- 닉네임만 켜짐
  - 닉네임 기준으로 묶음
- 둘 다 켜짐
  - 닉네임 + 메시지 기준으로 묶음

## 집계 규칙

### 도네

`1개 단위 금액` 기준으로 칸 수를 계산합니다.

예:

- 단위 금액 1000원
- 후원 금액 5000원
- 결과 5칸

### 채팅

- 금액은 0원
- 채팅 1회당 1칸

## 자동 종료 시간

둘 중 하나만 선택해서 사용할 수 있습니다.

- 카운팅 시간
- 몇 시까지

둘 다 끄는 것은 가능합니다.

수집 중에는 메인 화면 `현재 집계` 영역에서:

- 남은 시간
- 종료 기준

을 바로 확인할 수 있습니다.

종료되면 기준 문구가 `종료됨`으로 바뀝니다.

## 최근 들어온 후원

두 가지 방식으로 확인할 수 있습니다.

- 반영 결과
  - 실제 집계에 들어간 항목
- 원본 raw
  - 들어온 원본 이벤트

`로그 비우기`를 누르면 최근 후원과 raw 로그를 같이 비울 수 있습니다.

## 집계 결과 탭

집계 결과는 아래 탭으로 나눠 볼 수 있습니다.

- 전체
- 도네
- 채팅
- 필터 결과

## 필터 결과 탭

별도 후보 집합을 만드는 탭입니다.

설정 가능한 조건:

- 대상: 전체 / 도네 / 채팅
- 금액: 사용 안 함 / 이상 / 이하 / 정확히 / 구간
- 키워드: 사용 안 함 / 포함 / 시작 / 정확히

할 수 있는 것:

- 필터링된 결과만 보기
- `룰렛으로 이관`
- `전체 집계 복귀`

## 현재 집계 전체 삭제

메인 `현재 집계` 헤더에 있는 `전체 삭제`를 누르면:

- 현재 집계
- 최근 후원
- raw 로그
- 룰렛 대상

은 비워집니다.

하지만 유지되는 것:

- 최근 당첨 내역
- 저장된 룰렛
- 각종 설정

수집 중이면 삭제 후 그 시점부터 다시 새로 모으기 시작합니다.

## 제외된 항목

탭:

- 수동 제외
- 기준 제외

가능한 동작:

- 복원
- 삭제

기준 제외 탭에서 삭제하면 원본 데이터도 같이 정리됩니다.

## 저장된 룰렛

빠른 관리에서 할 수 있는 것:

- 새로 저장
- 현재 선택본 덮어쓰기
- 불러오기
- 삭제
- 선택 해제

필터 결과를 룰렛으로 이관한 상태도 함께 저장됩니다.

## 룰렛

### 돌리기

현재 룰렛 대상 기준으로 회전합니다.

### 연속

입력한 수만큼 연속 회전합니다.

예:

- `3`
- 당첨 기록: `1/3`, `2/3`, `3/3`

### 스킵

연속 회전 중 남은 결과를 한 번에 확정합니다.

### 순서 섞기

룰렛 항목의 배치 순서만 섞습니다.

### 전체화면

룰렛을 크게 띄워서 몰입감 있게 돌릴 수 있습니다.

## 최근 당첨

- 비우기 전까지 계속 누적됩니다.
- 스크롤로 이전 기록을 계속 볼 수 있습니다.

## 문제 확인 팁

### 방송 확인은 됐는데 실시간 반영이 안 될 때

- 운영 패널에서 수집 대상 확인
- 채팅 필터링 / 도네 필터링 조건 확인
- `원본 raw` 탭에 이벤트가 들어오는지 확인

### 최근 후원에는 보이는데 집계 결과에 안 보일 때

- 현재 탭이 `전체`인지 확인
- 필터 결과 상태인지 확인
- 묶기 기준 확인

### 시간 종료가 안 맞는 것처럼 보일 때

- `카운팅 시간`과 `몇 시까지`는 하나만 사용 가능
- 종료 문구는 `현재 집계` 영역에서 확인
